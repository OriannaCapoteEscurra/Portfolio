Arquitectura

La aplicación posee una arquitectura MVC. En el modelo usando el sistema de gestión de base de datos de MySQL PHP My Admin, la vista desarrollada con el Framework JavaScript Angular (en su versión 8); y el controlador desarrollado en el Framework de Phyton, Django.

Resumen

Una vez se inicia sesión correctamente, se presenta la sección de resumen, la cual muestra los ultimos movimentos en debito y credito, saldo disponible y el menu de navegación

Transferencias

Como toda app de banco moderno, esta app ofrece transferencias bancarias a cuentas del mismo banco, registradas o no, ofreciendo a estas ultimas la funcionalidad de ser registada al finalizar la operación.

Facturas

Para los usuarios no naturales se les facilita la funcionalidad de llevar un control sobre sus facturas pagadas, vencidas y vigentes. Asi como una sección de busqueda de facturas por rango de fechas

Aplicación

Estudiantil

Aplicación para la simulación de un banco y la interacción con usuarios finales (juridicos, naturales o gubernamentales) asi como la interacción con comercios para facilitar la compra venta mediante un medio electrónico.
